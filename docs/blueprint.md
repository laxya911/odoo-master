# **App Name**: Odoo Manager

## Core Features:

- Product Listing: Display a paginated list of products fetched from Odoo, with search and filtering.
- Partner Listing: Display a paginated list of partners (customers) fetched from Odoo, with search.
- POS Configuration Listing: Display a list of active POS configurations fetched from Odoo.
- POS Order Management: Display a paginated list of POS orders from Odoo, filterable by date range.
- Odoo API Integration: Connect to the Odoo JSON-2 API using API key authentication.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) for a professional and reliable feel.
- Background color: Light gray (#F5F5FA) to ensure readability and focus on the content.
- Accent color: Teal (#009688) to highlight key actions and elements.
- Body and headline font: 'Inter', a sans-serif font for a modern, neutral look suitable for both headlines and body text.
- Use clear and consistent icons from a library like FontAwesome or Material Icons to represent different entities and actions.
- Employ a clean, grid-based layout with consistent spacing and padding to maintain visual order.
- Subtle animations (e.g., fade-in effects, transitions) to enhance the user experience without being distracting.
